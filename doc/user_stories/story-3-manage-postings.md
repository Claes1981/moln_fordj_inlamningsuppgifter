# User Story 3: Manage Job Postings

**As an** administrator,
**I want to** create, edit, and deactivate job postings,
**so that** I can keep the portal's listings current and accurate.

## Acceptance Criteria

- An administrator can create a new job posting with title, location, description, and status.
- An administrator can edit the details of an existing job posting.
- An administrator can deactivate a posting (e.g., when the position is filled), removing it from candidate views.
- Only users with the administrator role can access the management interface.
- Changes are persisted to the database immediately.

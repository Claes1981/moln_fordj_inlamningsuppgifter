# User Story 2: Apply to a Job

**As a** candidate,
**I want to** submit an application for a specific job posting,
**so that** the employer can review my interest and qualifications.

## Acceptance Criteria

- A logged-in candidate can click an "Apply" button on a job detail page.
- The application form collects at minimum the candidate's name and a cover letter.
- The application is persisted in the database and linked to both the candidate and the job posting.
- The candidate receives confirmation that the application was submitted.
- A candidate cannot apply to the same job posting more than once.

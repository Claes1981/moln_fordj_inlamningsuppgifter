# User Story 1: Browse Job Listings

**As a** candidate,
**I want to** browse and search available job postings,
**so that** I can find positions that match my skills and preferences.

## Acceptance Criteria

- The landing page displays a list of all active job postings.
- Each listing shows at minimum the job title, location, and a short description.
- Clicking a listing navigates to a detail page with the full job description.
- Only postings marked as active/published are visible to candidates.
